Github Blog with Hexo - 글쓰기

Jan 22, 2022 in Github Blog

hexo로 쉽고 간편한 글쓰는 방법을 알려드리겠습니다.

목차

  1. Hexo 글쓰기
  2. more, excerpt
  3. Escape
  4. 이미지 삽입
  5. 로컬에서 확인하기
  6. 마치며

Hexo 글쓰기

hexo-cli의 간단한 명령으로 글을 만들 수 있습니다.

draft는 초안이라고 생각하시면 됩니다.

참고로 draft를 post로 만들면 상단 front-matter가 초기화되므로 확인하시기 바랍니다.

draft와 post는 scaffolds/post.md, scaffolds/draft.md를 템플릿으로 사용합니다.
템플릿을 수정하시려면 이 두 파일을 수정하시면 됩니다.
아래는 제가 사용하고 있는 템플릿입니다.

다른 것들은 보면 대충 뭔지 느낌이 올 겁니다. permalink만 설명할게요.
각 글의 주소는 http://호스트명/2022/01/22/Github-Blog-with-Hexo-글쓰기/ 과 같이 됩니다.
만약 이 부분을 변경하고 싶을 때 permalink를 지정해주면 설정한대로 적용됩니다.
front-matter를 참고하세요.

more, excerpt

more와 excerpt는 인덱스와 관련된 것들입니다.
글 작성 시에 사용하면 됩니다.

Escape

이렇게 템플릿 문법을 Escaping하여 문법을 그대로 보여지게 하고 싶다면 raw 구문을 이용합니다.

이미지 삽입

_config.yml파일이 post_asset_folder이 true로 되어있다면 글 제목과 같은 assets 디렉토리가 생깁니다.

그 디렉토리의 demo.png를 첨부하려면 위와같이 하면 됩니다.
Asset-Folders를 참고하세요.
